# `@ziiircom/components` > Theme

## global

### palette

theme.palette.primaryColor : color

theme.palette.secondaryColor : color

### fonts
theme.fonts : an array of string containing url to fonts

### font
theme.font : an object

font.size
font.style
font.weight
font.color

### spacings

theme.spacing: number default=8

theme.radius: number default=12

theme.smallRadius: number default=4

### shadow
theme.shadow: css shadow default=

## header
theme.header

theme.header.color : color (by default 'white')

theme.header.background : background css (by default use palette.primaryColor)

theme.header.shadow : shadow css (default=undefined)

theme.header.font : font object (default: use global one)
